# Peer review checklist

## Deterministic gates

- [ ] Every decisive factual claim has an opened supporting source.
- [ ] High-impact factual claims include a counter-evidence search record.
- [ ] Citations support the exact nearby claim.
- [ ] Independent sources do not merely repeat the same upstream source.
- [ ] Unresolved contradictions remain visible.
- [ ] The report separates facts, interpretations, values, and recommendations.
- [ ] The write target complies with active knowledge-base rules.

## Reviewer questions

1. What is the strongest unsupported or overstated claim?
2. Which material perspective or affected stakeholder is absent?
3. Is any disagreement manufactured or given false balance?
4. What evidence would most likely change the conclusion?
5. Does the recommendation follow from the evidence and the user's decision criteria?
